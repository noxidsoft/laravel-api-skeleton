# laravel-api-skeleton

`$ npm install && npm run dev`

`$ composer install`

`$ php artisan migrate`

`$ php artisan db:seed`

---

Run the application with the normal Laravel command
`$ php artisan serve`

Navigate to the standard Laravel localhost
<a href="http://127.0.0.1:8000" target="_blank">http://127.0.0.1:8000</a>
